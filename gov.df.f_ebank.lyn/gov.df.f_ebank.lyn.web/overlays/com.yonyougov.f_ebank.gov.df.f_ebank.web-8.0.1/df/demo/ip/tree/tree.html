树方法总结

  • 创建树

    （1）setting 配置

    （2）zTreeNode 节点数据

    (3)创建树

         例1 ： zTreeObj1 = $.fn.zTree.init($("#treeTest2"), setting1, zNodes1);

         例2 ：获取树 $("#treeTest3")[0]['u-meta'].tree

  • (1) 查找节点 ：

         a:根据节点数据的属性搜索

              getNodeByParam(key,value,parentNode)

         b:根据自定义规则搜索节点数据

              getNodesByFilter(filter, isSingle, parentNode, invokeParam)

         c:根据节点数据的属性搜索，获取条件模糊匹配的节点数据

              getNodesByParamFuzzy((key, value, parentNode))

  • (2)、折叠节点：

         expandAll(boolean) true 展开全部节点 false 合上全部节点

         展开 / 折叠 指定的节点 expandNode(treeNode, expandFlag, sonSign, focus, callbackFlag)

  • (3)、勾选节点：

         勾选/取消勾选 全部节点

              checkAllNodes（boolean） true 勾选 false 取消

         勾选/取消勾选 单个节点

              checkNode（treeNode, checked, checkTypeFlag, callbackFlag）

  • (4)、获取选中节点   getSelectNode(treeNode, addFlag, isSilent)

  • (5)、是否显示勾选框

         显示连接线在js里 setting.check.enable下设置

  • (6)、显示连接线的设置

    显示连接线在js里 setting.view.showLine下设置
