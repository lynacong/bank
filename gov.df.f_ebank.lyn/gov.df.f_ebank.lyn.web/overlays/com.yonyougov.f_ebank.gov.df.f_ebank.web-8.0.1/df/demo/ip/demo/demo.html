书写规范

HTML书写规范：http://codeguide.bootcss.com/

id与class :使用 "-" ,不建议使用 "_"（1、减少shift操作，2、良好区分js变量，IE6兼容性问题）。 如：iuap-web 而不是 iuap_web

CSS书写规范：http://codeguide.bootcss.com/

1、Positioning 位置属性 position, top, right, z-index, display, float等
2、Box model 盒模型 width、height、padding、margin等
3、Typographic 文字系列 font, line-height, letter-spacing, color- text-align等
4、Visual 视觉：边框、背景、动画 background、border、animation、 transition等
5.其他(animation, transition等)
布局： http://zh.learnlayout.com/

JavaScript书写规范
